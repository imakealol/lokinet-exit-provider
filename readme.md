# lokinet exit provider 

a (toy) vpn provider app server that takes ~~loki~~ oxen as payment.

## requirements

* cli wallet rpc
* postgresql
* python3
* python3-flask
* python3-psycopg2
* python3-requests
* python3-kubernetes
* [pyoxenmq](https://github.com/oxen-io/oxen-pylokimq)
* python3-gunicorn (optional)
* multipass (optional)
